## Project Setup Instructions

1. Clone the repository:

```bash
git clone https://github.com/your-username/your-repository.git
```

2. Install dependencies:

```bash
npm install
```

3. Run the project:

```bash
npm run dev
```

4. Build for production:

```bash
npm run build
```
